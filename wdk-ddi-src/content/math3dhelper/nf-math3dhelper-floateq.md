# FloatEq function


## -description

This function tests equality with FLOAT_TOLERANCE = 0.00001. x will be considered equal to y, if the difference between the two numbers is less than FLOAT_TOLERANCE.

## -parameters

### -param x

x value.

### -param y

y value.

## -returns

This function returns bool.
