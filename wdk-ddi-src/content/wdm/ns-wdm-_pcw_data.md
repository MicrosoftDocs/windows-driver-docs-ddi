# PCW_DATA structure


## -description

The `PCW_DATA` structure describes the data blocks that are associated with an instance. This structure is used with the [PcwAddInstance](nf-wdm-pcwaddinstance.md) and [PcwCreateInstance](nf-wdm-pcwcreateinstance.md) functions. Most developers do not need to access this structure directly because it is automatically managed by the Add/Create helper functions generated by the [CTRPP](/windows/win32/perfctrs/ctrpp) tool.

## -struct-fields

### -field Data

A pointer to a buffer of `Size` bytes containing a block of counter data.

### -field Size

A numeric value that specifies the size, in bytes, of the `Data` buffer.
