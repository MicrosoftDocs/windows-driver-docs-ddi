# _NFC_CX_DEVICE_MODE enumeration


## -description


Specifies device mode flags.


## -enum-fields




### -field NFC_CX_DEVICE_MODE_NCI

Enable NCI mode.


### -field NFC_CX_DEVICE_MODE_DTA

Enable DTA mode.


### -field NFC_CX_DEVICE_MODE_RAW

Enable RAW mode.
