# _WDI_IHV_TASK_PRIORITY enumeration


## -description

The WDI_IHV_TASK_PRIORITY enumeration defines IHV task priorities.

## -enum-fields

### -field WDI_IHV_TASK_PRIORITY_HIGH

High priority.

### -field WDI_IHV_TASK_PRIORITY_MEDIUM

Medium priority.

### -field WDI_IHV_TASK_PRIORITY_LOW

Low priority.
