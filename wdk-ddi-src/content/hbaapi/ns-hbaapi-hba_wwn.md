# HBA_wwn structure


## -description


The HBA_wwn structure contains a 64 bit world-wide name (WWN) that uniquely identifies an HBA. 


## -struct-fields




### -field wwn

Contains a 64 bit world-wide name (WWN) that uniquely identifies an HBA. .
