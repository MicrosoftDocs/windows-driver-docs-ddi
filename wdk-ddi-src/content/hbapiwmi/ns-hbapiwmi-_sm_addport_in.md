# _SM_AddPort_IN structure


## -description


The SM_AddPort_IN structure is used to provide input parameters to the SM_AddPort WMI method.


## -struct-fields




### -field PortWWN

A worldwide name (WWN) that indicates the port whose events are to be reported.


### -field EventType

An event type value.
