# D3D12DDI_RESOURCE_TRANSITION_BARRIER_0003 structure


## -description


Describes a transition barrier between subresources.


## -struct-fields




### -field hResource

The handle of a resource.


### -field Subresource

A subresource.


### -field StateBefore

The state before a transition.


### -field StateAfter

The state after a transition.
