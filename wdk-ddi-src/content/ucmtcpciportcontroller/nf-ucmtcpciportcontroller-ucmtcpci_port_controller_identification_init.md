# UCMTCPCI_PORT_CONTROLLER_IDENTIFICATION_INIT function


## -description

## -parameters




### -param Identification [out]

A pointer to the driver-allocated <b>UCMTCPCI_PORT_CONTROLLER_IDENTIFICATION</b> structure.


## -returns



This method does not return a value.
