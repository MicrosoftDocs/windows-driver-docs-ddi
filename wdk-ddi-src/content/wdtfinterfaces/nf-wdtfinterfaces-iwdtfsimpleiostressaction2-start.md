# IWDTFSimpleIOStressAction2::Start


## -description

Opens the device.

## -parameters

### -param pResult 

[out, retval]
True if the device opens; otherwise, false.

## -returns

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
