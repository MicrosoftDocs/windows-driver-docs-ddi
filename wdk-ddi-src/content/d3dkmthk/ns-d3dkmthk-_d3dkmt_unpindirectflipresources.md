# _D3DKMT_UNPINDIRECTFLIPRESOURCES structure

## -description

Arguments used to unpin direct flip resources.

## -struct-fields

### -field hDevice

[in] A handle to the device.

### -field ResourceCount

[in] The number of resources to unpin.

### -field pResourceList
 
[in] An array of resource handles to unpin.
