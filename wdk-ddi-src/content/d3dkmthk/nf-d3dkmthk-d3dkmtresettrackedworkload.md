# D3DKMTResetTrackedWorkload function


## -description

Resets the tracked workload to its initial state and resets all the rolling statistics.

## -parameters

### -param Arg1

[in] Pointer to a [D3DKMT_RESETTRACKEDWORKLOAD](ns-d3dkmthk-_d3dkmt_resettrackedworkload.md) structure.

## -returns


This function returns NTSTATUS.
