# D3DKMTPresentRedirected function


## -description

Redirects the present command.

Pointer to a [D3DKMT_PRESENT_REDIRECTED](ns-d3dkmthk-_d3dkmt_present_redirected.md) structure.

## -returns


This function returns NTSTATUS.
