# _D3DKMT_GETRUNTIMEDATA structure

## -description

Used to get runtime data.

## -struct-fields

### -field hAdapter

[in] A handle to the graphics adapter.

### -field hGlobalShare

[in] A handle to a global share.

### -field pRuntimeData

[in, out] Pointer to runtime data.

### -field RuntimeDataSize
 
[in] The runtime data size.
