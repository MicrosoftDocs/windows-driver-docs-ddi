# _TRISTATE enumeration


## -description

The <b>TRISTATE</b> enumeration indicates generic state values for true or false.

## -enum-fields

### -field TriStateUnknown

State is unknown.

### -field TriStateFalse

State is a false boolean value.

### -field TriStateTrue

State is a true boolean value.
