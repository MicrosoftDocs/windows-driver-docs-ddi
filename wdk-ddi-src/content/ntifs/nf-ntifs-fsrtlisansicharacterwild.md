# FsRtlIsAnsiCharacterWild macro


## -description

The <b>FsRtlIsAnsiCharacterWild</b> macro determines whether an ANSI character is a wildcard character.

## -parameters

### -param C

<p>Pointer to the character to be tested.</p>
