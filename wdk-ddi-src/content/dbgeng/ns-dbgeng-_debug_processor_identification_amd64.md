# _DEBUG_PROCESSOR_IDENTIFICATION_AMD64 structure


## -description


Identifies an AMD64 processor.


## -struct-fields




### -field Family

The family of the processor.


### -field Model

The model of the processor.


### -field Stepping

The stepping value of the processor.


### -field VendorString

A vendor specified string.
