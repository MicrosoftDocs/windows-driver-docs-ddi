# IDebugBreakpoint3::GetGuid


## -description


Returns a GUID for the breakpoint.


## -parameters




### -param Guid 
[out]
A unique ID returned for the breakpoint.


## -returns



If this method succeeds, it returns **S_OK**. Otherwise, it returns an HRESULT error code.
