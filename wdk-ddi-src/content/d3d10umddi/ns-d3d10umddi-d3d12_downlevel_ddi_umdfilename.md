# D3D12_DOWNLEVEL_DDI_UMDFILENAME structure

## -description

Contains the name of an OpenGL ICD that is based on the specified version of the DirectX runtime. 

## -struct-fields

### -field Filename
 
A string that contains the name of the OpenGL ICD.
