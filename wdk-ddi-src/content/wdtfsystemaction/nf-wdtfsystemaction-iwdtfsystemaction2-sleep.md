# IWDTFSystemAction2::Sleep


## -description

Puts the system into the desired sleep state.

## -parameters

### -param nSleepState 

[in]
The sleep state.

## -returns

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
