# CWiauDbgFn::~CWiauDbgFn method


## -description


The <b>CWiauDbgFn::~CWiauDbgFn</b> destructor is called when the function or method being traced by <b>CWiauDbgFn::CWiauDbgFn</b> is exited.


## -syntax


````
void ~CWiauDbgFn();
````


## -parameters






## -returns



This method does not return a value.
