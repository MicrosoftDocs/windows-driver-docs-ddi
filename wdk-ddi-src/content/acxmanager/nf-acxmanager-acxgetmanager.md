## -description

The **AcxGetManager** function returns the underlying ACXMANAGER handle.

## -parameters

### -param Context [in]

Reserved, set this to `NULL`.

## -returns

Returns the underlying ACXMANAGER handle.

## -remarks

The *Context* parameter must be set to `NULL`.
