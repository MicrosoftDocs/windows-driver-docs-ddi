# FWPS_FIELDS_EGRESS_VSWITCH_TRANSPORT_V4_ enumeration


## -description

  FWPS_LAYER_EGRESS_VSWITCH_TRANSPORT_V4 
  <a href="https://msdn.microsoft.com/en-us/library/windows/desktop/aa366492">run-time filtering layer</a>.
