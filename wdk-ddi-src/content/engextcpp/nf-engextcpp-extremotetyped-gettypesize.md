# ExtRemoteTyped::GetTypeSize method


## -description


The <b>GetTypeSize</b> method returns the size of the type represented by this object.


## -syntax


````
ULONG GetTypeSize();
````


## -parameters






## -returns



<b>GetTypeSize</b> returns the size, in bytes, of instance of the type.
