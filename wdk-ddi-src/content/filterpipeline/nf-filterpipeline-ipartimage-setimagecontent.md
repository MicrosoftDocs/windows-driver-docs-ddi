# IPartImage::SetImageContent


## -description

The <b>SetImageContent</b> method sets an image property that is based on the content type.

## -parameters

### -param pContentType

#### - contentType [in]

The type of content of the image.

## -returns

<b>SetImageContent</b> returns an <b>HRESULT</b> value.
