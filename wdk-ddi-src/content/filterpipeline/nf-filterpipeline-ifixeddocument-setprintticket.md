# IFixedDocument::SetPrintTicket


## -description

The <b>SetPrintTicket</b> method inserts a print ticket into the fixed document.

## -parameters

### -param pPrintTicket 

[in]
A pointer to the print ticket to be inserted.

## -returns

<b>SetPrintTicket</b> returns an <b>HRESULT</b> value.
