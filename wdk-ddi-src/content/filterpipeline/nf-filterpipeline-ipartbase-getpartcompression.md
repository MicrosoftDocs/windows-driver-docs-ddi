# IPartBase::GetPartCompression


## -description


The <b>GetPartCompression</b> method gets the compression of the part.


## -parameters




### -param pCompression 
[out]
A pointer to the compression option data for the part.


## -returns



<b>GetPartCompression</b> returns an <b>HRESULT</b> value.
