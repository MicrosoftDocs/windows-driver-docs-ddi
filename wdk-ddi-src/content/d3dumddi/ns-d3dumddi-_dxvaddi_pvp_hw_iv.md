# _DXVADDI_PVP_HW_IV structure


## -description


The DXVADDI_PVP_HW_IV structure contains two 64-bit values that combine to form a 128-bit protected video path (PVP) value.


## -struct-fields




### -field IV

[in] A 64-bit initialization vector.


### -field Count

[in] A 64-bit number.
