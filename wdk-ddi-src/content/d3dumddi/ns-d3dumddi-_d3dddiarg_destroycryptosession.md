# _D3DDDIARG_DESTROYCRYPTOSESSION structure


## -description


The D3DDDIARG_DESTROYCRYPTOSESSION structure contains the handle to an encryption session that is destroyed in a call to the <a href="https://docs.microsoft.com/windows-hardware/drivers/ddi/content/d3d10umddi/nc-d3d10umddi-pfnd3d11_1ddi_destroycryptosession">DestroyCryptoSession</a> function. 


## -struct-fields




### -field hCryptoSession

[in] The handle to the encryption session that the driver destroys.
