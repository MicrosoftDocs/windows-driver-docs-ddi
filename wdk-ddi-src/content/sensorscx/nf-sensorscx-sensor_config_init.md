# SENSOR_CONFIG_INIT function


## -description

## -parameters

### -param pConfig 

[out]
A pointer to a new <a href="https://docs.microsoft.com/windows-hardware/drivers/ddi/sensorscx/ns-sensorscx-_sensor_config">SENSOR_CONFIG</a> structure.
