# __MIDL_IKeywordDetectorOemAdapter_0003 structure

The <b>KEYWORDSELECTOR</b> struct is a triplet of IDs that uniquely select a particular keyword, language, and user combination.

### -field KeywordId

Identifies a keyword function.

### -field LangId
