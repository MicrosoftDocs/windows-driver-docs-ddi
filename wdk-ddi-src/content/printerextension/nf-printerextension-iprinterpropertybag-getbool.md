# IPrinterPropertyBag::GetBool


## -description


Reads a specified boolean property.


## -parameters




### -param bstrName [in]

The name of the property.


### -param pbValue [out, retval]

The returned property value.


## -returns



This method returns an <b>HRESULT</b> value.
