# IPrintSchemaCapabilities::GetOptions


## -description


Gets all the options of a feature.


## -parameters




### -param pFeature [in]

The specified feature.


### -param ppOptionCollection






#### - ppOptions [out, retval]

The returned options.


## -returns



This method returns an <b>HRESULT</b> value.
