# tagSMS_MESSAGE_TYPES enumeration


## -description



The <b>SMS_MESSAGE_TYPES</b> enumeration type describes the content type of a short message service (SMS) message.




## -enum-fields




### -field SMS_TEXT_MESSAGE

A text message.


### -field SMS_BINARY_MESSAGE

A binary message.


## -remarks



This enumeration is used by the <a href="https://docs.microsoft.com/windows/desktop/wpd_sdk/wpd-command-sms-send-command">WPD_COMMAND_SMS_SEND Command</a>.
