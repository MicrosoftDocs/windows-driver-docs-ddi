# TapeClassLiDiv function


## -description


The <b>TapeClassLiDiv</b> routine performs a division of the two indicated integers. 


## -parameters




### -param Dividend [in]

Contains the dividend.


### -param Divisor [in]

Contains the divisor.


## -returns



<b>TapeClassLiDiv</b> returns the result of the division.
