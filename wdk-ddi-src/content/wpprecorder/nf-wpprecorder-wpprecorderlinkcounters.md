# WppRecorderLinkCounters macro


## -description

The <b>WppRecorderLinkCounters</b> method  uses a sequence number to merge logs captured in different buffers by a driver.

## -parameters

### -param CounterOwner 

[in]
ID of the counter whose current value is to be read.
