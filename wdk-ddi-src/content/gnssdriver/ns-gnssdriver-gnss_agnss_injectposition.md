# GNSS_AGNSS_INJECTPOSITION structure


## -description


This structure defines the format for AGNSS position injection.


## -struct-fields




### -field Size

Structure size.


### -field Version

Version number.


### -field Age

Indicates how long the position has been aged in seconds.


### -field BasicData

Position data.


### -field AccuracyData

Position accuracy.
