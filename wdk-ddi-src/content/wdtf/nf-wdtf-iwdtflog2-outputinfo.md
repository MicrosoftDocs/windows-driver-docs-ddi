# IWDTFLOG2::OutputInfo


## -description


Writes an informational entry to the test case log.


## -parameters




### -param sInfoString 
[in]
The information string.


## -returns



If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
