# IWDTFNumbers2::Remove


## -description

Removes a number from the collection.

## -parameters

### -param Index 

[in]
The 0-based index value of the number to remove.

## -returns

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
