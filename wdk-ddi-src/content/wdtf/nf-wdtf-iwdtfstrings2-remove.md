# IWDTFStrings2::Remove


## -description

Removes a string from the collection.

## -parameters

### -param Index 

[in]
The 0-based index value of the string to remove.

## -returns

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
