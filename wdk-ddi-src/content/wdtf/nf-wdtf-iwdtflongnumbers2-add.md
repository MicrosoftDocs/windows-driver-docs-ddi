# IWDTFLongNumbers2::Add


## -description

Adds a single long number to the collection.

## -parameters

### -param Number 

[in]
The long number to add to this collection.

## -returns

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
