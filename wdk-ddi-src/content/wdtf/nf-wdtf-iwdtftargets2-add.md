# IWDTFTargets2::Add


## -description

Add a single item to the collection.

## -parameters

### -param pTarget 

[in]
The item to add to this collection.

## -returns

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
