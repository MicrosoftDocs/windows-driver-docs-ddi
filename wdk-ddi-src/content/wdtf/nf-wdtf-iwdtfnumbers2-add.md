# IWDTFNumbers2::Add


## -description

Adds a single number to the collection.

## -parameters

### -param Number 

[in]
The number to add to this collection.

## -returns

If this method succeeds, it returns **S_OK**. Otherwise, it returns an **HRESULT** error code.
